# Button

Simple button<br />

<br />

## Props:

- `rootTagAttributes`
    - **Button** tag attributes<br /><br />

- `refApi`

- `className`

- `type`
    - **Button** type
    - **String**
    - Default is **'button'**<br /><br />
 
- `value`
    - **Button** content
    - **React.ReactNode**<br /><br />

- `disabled`
    - Indicates whether a **Button** is disabled
    - **Boolean**<br /><br />

- `onClick`
    - Click handler
    - **Function**. Has 1 argument:
        - **event** - **React.MouseEvent**
