# Button

Simple button<br />

<br />

## Props:

- `refApi`
    - Component root reference params<br /><br />

- `className`
    - Root element class name
    - **String**<br /><br />

- `type`
    - **Button** type
    - **String**
    - Default is **'button'**<br /><br />
 
- `value`
    - **Button** content
    - **React.ReactNode**<br /><br />

- `disabled`
    - Disables **Button**
    - **Boolean**<br /><br />

- `onClick`
    - Click handler
    - **Function**. Has 1 argument:
        - **event** - **React.MouseEvent**<br /><br />

- `rootTagAttributes`
    - **Button** root tag attributes<br /><br />
